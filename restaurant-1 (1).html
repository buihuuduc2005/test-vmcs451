🔥 Kết nối Firebase Realtime

Để đơn hàng và điểm danh sync real-time giữa các máy, cần Firebase Realtime Database (miễn phí). Chỉ cần setup một lần.

  1. Vào console.firebase.google.com → Tạo project
  2. Realtime Database → Create → Start in test mode
  3. ⚙️ Project Settings → </> Web app → copy firebaseConfig
  4. Dán vào ô dưới → Kết nối
Bỏ qua — Dùng chế độ local (1 máy, không sync)
